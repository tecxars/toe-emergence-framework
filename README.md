
# Motivation: Information Storage as an Abstract Foundation

## Starting Point

Suppose the fundamental level of the universe does not consist of continuous quantities, but of a finite set of distinguishable informational contents. This assumption is compatible with theoretical limits of information storage and information processing. It suggests that nature does not represent arbitrarily fine values, but that every physically realizable structure is effectively information‑limited.

If one takes this perspective seriously, an abstract picture emerges:
The universe behaves like an information storage system whose contents are accessible, but not arbitrarily retrievable in parallel.

## Information Storage and Structure

An information storage system requires internal organization. This can be understood abstractly as a set of states between which certain operations are allowed. The structure of such states can be described by algebraic relations that ensure the storage remains closed when operations are performed on it.

In such a system, it is natural that not all operations can be executed simultaneously. Some accesses influence each other, others are only possible sequentially. These limitations are not a weakness but a structural property of the storage system.

## Limited Access

Access to information is not free. Every query requires resources. A fundamental result of information processing shows that even elementary operations possess a minimal energy cost. This suggests that unlimited parallel access to all informational contents is not realizable.

Instead, access is structured:

- Some areas of the storage cannot be read simultaneously.
- The order of accesses influences the results.
- There is no global view of all information at once.

These properties are typical for systems whose internal structure is not arbitrarily accessible in parallel.

## Abstract Organization

An information storage system that allows only limited access requires a form of organization that ensures operations remain consistent. This leads to an abstract structure in which:

- states are interconnected,
- operations remain closed,
- and the totality of possible accesses forms a stable unit.

Such structures can be described mathematically with precision, without presupposing physical concepts such as space, time, or metric. The organization of the storage system itself provides the foundation for later interpretations.

## Emergence of Higher Structures

If access is the fundamental operation, more complex concepts can be derived from it:

- Distinguishability of states can serve as the basis of a metric.
- Stability of accesses can define a topology.
- Scaling behavior can generate a dimension.
- Dependencies between accesses can form a directed structure.

These concepts do not arise from assumptions but from analyzing the access rules of an information‑limited organized system.

## Perspective

From this viewpoint, the universe is not a continuous geometric object but an information storage system with limited access. The fundamental level consists of states and operations that form a closed structure. All higher properties — including geometric concepts — can be understood as emergent effects of this organization.

This perspective is abstract, mathematically controllable, and independent of concrete physical models. It offers a possible starting point for a theory in which spacetime is not presupposed but emerges from the structure of the information storage system.

# Why a C*-Algebra Is the Natural Mathematical Structure

If one takes the perspective described above seriously, a specific mathematical form follows: an information‑limited storage system possesses exactly the properties described by a C*-algebra.

Such a storage system satisfies:

1. Operations are bounded.  
   Physically realizable processes have limited effect. This corresponds to bounded operators.

2. States are expectation‑value assignments.  
   A state is a positive, normalized linear functional. This precisely describes the state theory of a C*-algebra.

3. Operations must be closed.  
   The composition of accesses must not leave the system. This leads to an algebraic structure closed under multiplication and involution.

4. Noncommutativity arises from limited access.  
   If not all information can be retrieved simultaneously, the result depends on the order of operations. This is the natural origin of noncommutative structures.

Thus:

The C*-algebra is not an additional assumption but the precise mathematical form of an information‑limited storage system. It represents states, operations, local accesses, dynamics, and limitations of parallelizability in a single consistent structure.

This algebraic foundation makes it possible for all higher concepts — metric, topology, dimension, causality — to emerge from the information structure itself, without presupposing them.

# Graph Structure as an Organizational Form

An information‑limited storage system requires not only an algebraic description but also an organizational form that determines which operations can interact locally. This organization can be modeled by a graph structure.

A graph describes:

- which information cells are connected,
- which operations act locally,
- how far an access can propagate,
- and which areas are independent of each other.

The nodes represent elementary information cells.  
The edges represent possible local interactions or accesses.

Such a graph structure is not geometric in the classical sense but an abstract dependency structure. It determines which operations can be executed simultaneously and which influence each other. Thus, it determines the form of local dynamics and the range of influences.

In combination with the C*-algebra, a consistent picture emerges:

- The C*-algebra describes the algebraic structure of operations.
- The graph describes the local coupling structure of these operations.
- The dynamics arise as compositions of local updates along the edges.
- The emergent geometry arises from analyzing these local structures.

The graph structure is therefore not an additional physical object but the abstract organizational form of the information storage system itself. It enables the definition of locality, causality, and influence regions without presupposing space or time.

This combination of C*-algebra and graph structure forms the foundation for the further development of the theory.

This is exactly what this project is built upon.
